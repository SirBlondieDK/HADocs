# HADocs Home Assistant Add-on

This experimental add-on runs HADocs directly from Home Assistant.

The first MVP:

- connects through the Home Assistant Supervisor API
- generates documentation locally
- writes reports to `/share/hadocs`
- does not upload data to the cloud

## Usage

1. Install the add-on.
2. Choose a project name.
3. Start the add-on.
4. Open `/share/hadocs` using Studio Code Server, Samba or File editor.

This is an early development build.
