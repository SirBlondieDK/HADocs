# Screenshot rename guide

Copy your real PNG screenshots into `assets/screenshots/` using these names:

| Screenshot | Filename |
|---|---|
| Main dashboard | `01-dashboard-overview.png` |
| Installation Overview / Health Score v2 | `02-installation-overview.png` |
| Top Recommendation / repair plan | `03-top-recommendations.png` |
| Root Cause cards | `04-root-cause-analysis.png` |
| Score explanation / history | `05-health-score-history.png` |
| Generated output links | `06-generated-output.png` |
