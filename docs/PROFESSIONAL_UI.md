# Professional UI

This update improves the HADocs desktop GUI.

## Added

- Dark theme matching the generated HTML dashboard.
- Sidebar navigation.
- Dashboard-like metric cards.
- Better console styling.
- Improved first-run wizard.
- Improved settings and about dialogs.
- Better scan status and progress display.

## Scope

This is still pure Tkinter/ttk, so it remains lightweight and easy to package with PyInstaller.

No cloud, telemetry, analytics or AI calls are added.
