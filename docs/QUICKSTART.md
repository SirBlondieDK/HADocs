# Quick Start

For a very simple guide, start here:

```text
docs/BEGINNER_GUIDE.md
```

## Short version

1. Install dependencies:

```powershell
py -3.14 -m pip install -r requirements.txt
```

2. Run tests:

```powershell
py -3.14 -m pytest
```

3. Start HADocs:

```powershell
py -3.14 main.py
```

4. Enter:

- Home Assistant URL
- Long-Lived Access Token
- Project name

5. Generate documentation.

6. Open:

```text
output/index.html
```

## Recommended first checks

Start with:

- Executive Dashboard
- Root Causes
