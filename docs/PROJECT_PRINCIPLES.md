# HADocs Project Principles

## Mission

Help every Home Assistant user understand, maintain and improve their smart home — without compromising privacy.

## Principles

1. **Privacy First** - no data leaves the user's computer by default.
2. **Read Only** - HADocs never calls services or changes Home Assistant.
3. **Local First** - reports are generated locally.
4. **Documentation First** - analysis depends on accurate documentation.
5. **Human First** - explain causes in human language, not just entity counts.
6. **AI Compatible** - structured exports are optional and user-controlled.
