# Privacy

HADocs is built for privacy-conscious Home Assistant users.

| Item | Status |
|---|---|
| Cloud upload | No |
| Telemetry | No |
| Analytics | No |
| Tracking | No |
| AI calls | No |
| Local report generation | Yes |

HADocs may read Home Assistant states, entities, devices, areas, integrations, labels and metadata.

HADocs writes local reports such as HTML, Markdown, CSV and optional knowledge exports.

Reports may include private device names, room names, entity names and IP addresses. Review reports before sharing.
