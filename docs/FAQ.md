# Frequently asked questions

## Does HADocs require internet access?

No. HADocs analyzes Home Assistant locally. Network access to the Home Assistant instance is still required.

## Does HADocs upload Home Assistant data?

No. HADocs does not upload installation data, telemetry, or analytics.

## Does HADocs use external AI?

No. Analysis is deterministic and evidence-based. Knowledge Packs can be shared manually, but HADocs never uploads them automatically.

## Does HADocs modify Home Assistant?

No. HADocs is read-only and does not call services or change entities, automations, dashboards, or devices.

## Is Python required?

Not for the Windows release. Source development requires Python 3.11 or newer; the current development commands use Python 3.14.

## Where should I begin after a scan?

Start with the Executive Summary and Smart Recommendations. Then use Root Cause Analysis and Explorer to inspect affected devices, entities, integrations, and evidence.

Continue with the [Quick Start](QUICKSTART.md), or return to the [documentation home](README.md).
