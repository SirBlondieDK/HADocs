# v0.14 Intelligence Foundation

Adds the first v0.14 Smart Home Intelligence features.

## Added

- Health Forecast
- Root Cause Impact Score
- Local Explain This engine
- Dashboard Health Forecast section
- Explain This details on Root Cause cards
- Impact badges on Root Cause cards

## Apply

```powershell
py -3.14 tools/apply_v014_intelligence_foundation.py
py -3.14 -m pytest
py -3.14 main.py
```

No cloud calls. No external AI. Everything is local and rule-based.
