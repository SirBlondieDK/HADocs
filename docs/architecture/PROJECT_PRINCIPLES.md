# HADocs project principles

## Mission

Help Home Assistant users understand installation health, identify root causes, and prioritize improvements without compromising privacy.

## Principles

1. **Privacy First** — no data leaves the user's infrastructure automatically.
2. **Local First** — analysis and exports are generated locally.
3. **Read Only** — HADocs never calls services or changes Home Assistant.
4. **Evidence Before Assumptions** — findings must be supported by collected data.
5. **Root Causes Before Symptoms** — group related incidents into actionable causes.
6. **Explainable Prioritization** — Health Score and Smart Recommendations must show why work matters.
7. **User-controlled sharing** — Knowledge Packs and exports are shared only by the user.
