# Analysis

HADocs converts collected Home Assistant data into deterministic, evidence-based findings. The Analysis page connects the Executive Summary, Smart Recommendations, Root Cause Analysis, and Health Score details in one interactive view.

Analysis is read-only, runs locally, and does not rely on external AI services.

## Analysis flow

```text
Collected data → Effective incidents → Root Cause Analysis → Health Score → Smart Recommendations
```

Start with the Executive Summary, then open a recommendation or root cause to inspect affected devices, entities, integrations, and supporting evidence.

Continue with [Root Cause Analysis](Root-Causes.md), or return to the [documentation home](../../README.md).
