# Exports

HADocs creates portable local output for review, archiving, and user-controlled sharing.

Available output includes:

- Interactive HTML Dashboard
- HTML Explorer
- Markdown reports
- CSV exports
- Knowledge Pack and redacted Knowledge Pack
- History snapshots

Exports may contain private Home Assistant data. Review them before sharing and keep generated output out of public commits.

Review [Privacy](../../privacy.md) and [Knowledge Pack](Knowledge-Pack.md), or return to the [documentation home](../../README.md).
