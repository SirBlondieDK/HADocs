# Explorer

Explorer is an analysis tool for understanding relationships across areas, devices, entities, integrations, and findings.

Use Explorer to move from a root cause to affected objects, inspect installation context, and follow related evidence. It complements the Dashboard Engine by providing a detailed, searchable view of the installation model rather than a file browser.

See also [Device Overrides](Device-Overrides.md) and [Knowledge Pack](Knowledge-Pack.md), or return to the [documentation home](../../README.md).
