# Knowledge Pack

The Knowledge Pack is a structured, locally generated representation of HADocs analysis results. It can provide context to another person or an AI assistant, but HADocs never uploads it automatically.

```text
output/knowledge/
output/knowledge/redacted/
```

You decide whether either version is shared. Prefer the redacted Knowledge Pack when external review is necessary, and inspect it before distribution.

See [Privacy](../../privacy.md) before sharing, or return to the [documentation home](../../README.md).
