# Desktop GUI

The desktop application is the recommended way to use HADocs.

## Sidebar

- Scan Home Assistant
- Dashboard
- Explorer
- Markdown
- Output Folder
- Doctor
- Settings
- About

## Dashboard

Shows:

- Health Score
- Top Recommendation
- Installation KPIs
- Scan Status

## Developer Log

Accessible from 'Open developer log' and opens in its own window.

## Workflow

1. Enter Home Assistant URL
2. Paste Long-Lived Access Token
3. Press Scan
4. Open Dashboard
