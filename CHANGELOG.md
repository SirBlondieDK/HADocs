# Changelog

All notable changes to HADocs are documented here.

## v0.11.2 - Beginner Guide Release

### Added
- Beginner-friendly guide: `docs/BEGINNER_GUIDE.md`.
- Non-technical quick start wording in `README.md`.
- Troubleshooting section for the most common first-run problems.
- Simple explanation of Home Assistant tokens.
- Simple explanation of what files to open after generating a report.
- Child-friendly "what to click" workflow.

### Changed
- README now links directly to the beginner guide near the top.
- Quick start documentation is now easier for non-technical users.

### Documentation
- Added a guide written for users with little or no terminal/Git/Python experience.
- Added copy/paste commands for Windows.
- Added "what not to worry about" section.

## v0.11.1 - Stability & Test Environment

### Added
- Local pytest temp folder configuration.
- Test environment documentation.
- Cleanup script for local test/cache artifacts.

### Fixed
- Pytest setup errors on Windows when `AppData\\Local\\Temp\\pytest-of-USER` is locked or inaccessible.
